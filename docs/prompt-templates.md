# Prompt Templates

このドキュメントは、画像支給ベースで LP 実装を進めるときの定型プロンプト集です。

## 1. 初回プロンプト

```txt
このプロジェクトは MakeShop 用 LP の Astro スターターです。
作業前に以下のドキュメントを読んで前提を合わせてください。

- README.md
- docs/coding-rules.md
- docs/design-handoff-flow.md

この案件は Figma MCP 前提ではなく、画像支給ベースで進めます。
実装は semantic HTML / FLOCSS + BEM / Core Web Vitals 重視で進めてください。

まずはドキュメントを読んで、今回の進め方を理解してください。
理解後は、どの順番で作業するかを短く整理してください。
```

## 2. 全体画像を渡すときのプロンプト

```txt
これから PC / SP の全体画像を渡します。
まずは全体画像をもとに、ページ全体の骨組みと大まかなレイアウトを実装してください。

この段階でやってよいこと:
- セクション分割
- semantic HTML の構造化
- 見出し階層の整理
- コンテナ幅とレイアウト設計
- 背景色の切り替え
- おおまかなタイポ設計
- PC / SP の大枠の組み方

この段階ではまだ詰めすぎないこと:
- 細かい余白
- 正確な文字サイズ
- 装飾の微調整
- absolute 配置の細部
- hover / animation
- 本画像の差し込み

画像は最初はプレースホルダで進めてください。
色とフォントは既存 token 設計に従ってください。
```
